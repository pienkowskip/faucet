# Faucet

Crypto-currency faucets bot. Uses selenium, Xvfb & external captcha solving service to headlessly claim
crypto-currency from web faucet.

## Installation

Add this line to your application's Gemfile:

    gem 'faucet'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install faucet
